# This folder contains all data used in the manuscript
## Enhancer network and Initial features
- **EnhNetG.txt**: an enhancer similarity network with initial features constructed from enhancer target genes
- **enh2disease-1.0.2.txt.bed.fasta**: enhancer sequences
  
## Disease similarity networks
- **DODisSimNet.txt** (i.e., DiSimNet): a disease similarity network derived from the Disease Ontology vocabulary


## Known disease–enhancer associations  
- **EDRelation.csv**: obtained from the DiseaseEnhancer database

## Others
- **enh2target-1.0.2-Entrez.txt**: enhancer–target gene relationships were obtained from the DiseaseEnhancer database
